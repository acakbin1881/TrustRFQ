# Copy and Terminology Rule

Use TrustRFQ-specific OTC settlement language.

Use:

- RFQ
- private RFQ
- quote
- accepted quote
- deal
- settlement
- counterparty
- escrow
- funds locked
- settlement sent
- settlement confirmed
- release
- dispute
- contract ID

Avoid unless explicitly discussing non-goals:

- freelancer
- project
- invoice
- payroll
- task board
- marketplace listing
- order cart
- checkout
- subscription
- creator campaign
- auction
- outbid
- public bidding
- swap
- AMM

Copy should make the trust problem obvious: who is waiting on whom, whether funds are locked, what action comes next, and what proof exists.
